# Description du projet
SimpleAlarm est une application de gestion d'alarmes qui permet aux utilisateurs de créer, gérer et personnaliser leurs alarmes selon leurs besoins.

## Liste des fonctionnalités
- Créer des alarmes personnalisées
- Modifier ou supprimer des alarmes
- Notifications sonores et visuelles
- Interface utilisateur simple et intuitive

## Instructions d'installation
1. Clonez le dépôt : `git clone https://github.com/jeanfumeplus-tech/SimpleAlarm.git`
2. Accédez au répertoire du projet : `cd SimpleAlarm`
3. Installez les dépendances : `npm install`
4. Démarrez l'application : `npm start`

## Guide d'utilisation
- Pour créer une nouvelle alarme, cliquez sur le bouton "Ajouter une alarme".
- Remplissez les informations requises et cliquez sur "Enregistrer".
- Pour modifier une alarme, sélectionnez-la et changez les réglages selon votre convenance.
- Pour supprimer une alarme, utilisez l'option "Supprimer" dans le menu de l'alarme.
